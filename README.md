# ResúMatch AI

An AI-powered career intelligence platform that helps job seekers understand how well their resumes align with real-world opportunities.

ResúMatch AI goes beyond traditional job boards by analyzing resumes, matching candidates against live job listings, identifying skill gaps, generating ATS scores, recommending career paths, and providing salary intelligence.

---

## Features

### Resume Intelligence

* PDF resume upload
* Automatic skill extraction
* Experience and profile analysis
* Candidate profile generation

### AI Job Matching

* Live job ingestion
* Match score calculation
* Ranked job recommendations
* Match explanation engine

### Career Intelligence

* ATS Resume Score
* Skill Gap Analysis
* Resume Improvement Suggestions
* Career Recommendations
* Salary Intelligence

### Dashboard

* Modern React dashboard
* Saved Jobs functionality
* Candidate profile view
* Career insights interface

## How It Works

1. Upload a resume PDF.
2. Extract candidate information and skills.
3. Search live jobs based on profile keywords.
4. Score each job against the candidate profile.
5. Generate career intelligence insights.
6. Present results through an interactive dashboard.
